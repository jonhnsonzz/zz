# 高新认定自检工具

> High-Tech Enterprise Qualification Self-Check Tool

> **English:** A free online tool to check if your company qualifies for China's High-Tech Enterprise (HTE) certification. 6 questions, 3 minutes, get a score + action plan. No data upload, all processing in your browser.

> **中文：** 一个免费在线的自检工具，帮你快速判断企业是否符合高新技术企业认定条件。不符合也给行动指南——告诉你缺什么、怎么补、流程是什么。

## 怎么用？

1. 打开 https://jonhnsonzz.github.io/zz/
2. 6个问题，每个下拉选择你的实际情况
3. 点「开始评估」
4. 系统自动打分，分三级：
   - **A级（85分以上）** → 条件充分
   - **B级（70-84分）** → 基本达标
   - **C级（70分以下）** → 暂不满足
5. **逐项诊断**：每个维度标注 ✅ / ⚠️ / ❌，不达标项给出具体补救方案
6. **行动指南**：高新认定全流程（准备→提交→评审→拿证），不管符不符合都能带走

## 评估维度

| 维度 | 满分 | 说明 |
|------|------|------|
| 成立时间 | 10分 | 需注册满1年以上 |
| 知识产权 | 15分 | I类知识产权（发明专利等）数量 |
| 科技人员占比 | 15分 | 科技人员占职工总数≥10% |
| 研发费用比例 | 20分 | 近三年研发费用占销售收入比例达标 |
| 高新收入占比 | 20分 | 高新收入占同期总收入≥60% |
| 创新能力评价 | 20分 | 知识产权+成果转化+研发管理+成长性≥70分 |

## 隐私说明

- 所有数据仅在浏览器本地计算，不上传任何服务器
- 不收集任何个人信息

## 技术说明

纯前端HTML单页应用，零依赖，可直接在浏览器中运行。
