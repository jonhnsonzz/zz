# 高新认定自检工具

> 6个问题 · 3分钟快速评估 · 数据不上传服务器

## 这是什么？

一个免费在线的自检工具，帮你快速判断企业是否符合 **高新技术企业认定** 条件。不用注册、不用填隐私信息，所有计算都在你浏览器里完成。

## 怎么用？

1. 打开 https://jonhnsonzz.github.io/zz/
2. 6个问题，每个下拉选择你的实际情况
3. 点「开始评估」
4. 系统自动打分，告诉你：
   - **A级（85分以上）** → 条件充分，建议立即申报
   - **B级（70-84分）** → 基本满足，补强后申报
   - **C级（70分以下）** → 差距较大，建议培育后申报
5. 每个维度有 ✅ / ⚠️ / ❌ 标记，一目了然哪些达标哪些要补
6. 底部有下一步建议

## 评估维度

| 维度 | 满分 | 说明 |
|------|------|------|
| 成立时间 | 10分 | 需注册满1年以上 |
| 知识产权 | 15分 | I类知识产权（发明专利等）数量 |
| 科技人员占比 | 15分 | 科技人员占职工总数≥10% |
| 研发费用比例 | 20分 | 近三年研发费用占销售收入比例达标 |
| 高新收入占比 | 20分 | 高新收入占同期总收入≥60% |
| 创新能力评价 | 20分 | 知识产权+成果转化+研发管理+成长性≥70分 |

## 隐私说明

- 所有数据仅在浏览器本地计算
- 不上传任何服务器
- 不收集任何个人信息

## 技术说明

纯前端HTML单页应用，零依赖，可直接在浏览器中运行。
